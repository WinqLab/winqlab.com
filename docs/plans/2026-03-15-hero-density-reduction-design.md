# WinqLab Hero Density Reduction Design

**Date:** 2026-03-15
**Status:** Approved

## Goal

Reduce visual crowding in the homepage hero so the first screen feels more mature, calmer, and easier to read while keeping the product-like technology tone.

## User Feedback

1. The hero still feels unattractive and crowded.
2. The main headline is too large and breaks too aggressively.
3. Supporting text spans too wide and adds pressure to the layout.
4. The right-side card still competes with the main message.
5. English labels in the header and hero add noise.

## Approved Direction

Use a left-led editorial hero with a quiet supporting card on the right.

### Headline Direction

Keep the approved core message, but render it as a strict two-line title:

`从研究假设到系统验证`

`再到实盘交易`

The title should be smaller than the current version, with looser line-height and more negative space around it.

### Layout Direction

1. Keep a two-column layout.
2. Make the left column the clear focus area.
3. Narrow the text column width so paragraphs feel more controlled.
4. Shrink the right-side card and place it slightly lower so it reads as support, not competition.
5. Reduce top chrome density by removing English-heavy header labels and badges.

### Card Direction

1. Keep only one supporting card.
2. Remove long stacked paragraph blocks from the card.
3. Use a short card title and a few compact information rows.
4. Make the card quieter with more padding and less headline weight.

### Tone

The hero should feel like a polished technology product homepage rather than a dashboard screenshot or terminal interface mockup.
